WRONG + WRONG = RIGHT

Saturday, January 18, 2020, 8:00:00 am

Solve the following puzzle, where O = 0 and no two distinct letters stand for the same integer.

  W R O N G
+ W R O N G
------------
  R I G H T

First, let's replace O by 0. Then

  W R 0 N G
+ W R 0 N G
------------
  R I G H T

Since 'NG' <= 99, we see that 'GHT' <= 198. Since G is not 0, G must be 1. Then

  W R 0 N 1
+ W R 0 N 1
------------
  R I 1 H 2

It is not hard to see N >= 5. H is of course 2N - 10. H cannot be 0, 1 or 2, so H = 2N - 10 >= 3, which implies that N is 7, 8 or 9.

Note that 'WR' + 'WR' = 'RI'. This means 20W + 2R = 10R + I, or I = 20W - 8R. Hence I is 4 or 8.

If I is 4, then 1 = 5W - 2R -> W = 1 + 2t, R = 2 + 5t. R must be 7 -> W must be 3. Then

  3 7 0 N 1
+ 3 7 0 N 1
------------
  7 4 1 H 2

Now N cannot be 7. If N = 8, then H = 6; if N = 9, then H = 8. Both are perfectly acceptable solutions.

  3 7 0 8 1                     3 7 0 9 1
+ 3 7 0 8 1         or        + 3 7 0 9 1
------------                  ------------
  7 4 1 6 2                     7 4 1 8 2

If I is 8, then 2 = 5W - 2R -> W = 2 + 2t, R = 4 + 5t. W cannot be 2, so W = 4, R = 9. Then

  4 9 0 N 1
+ 4 9 0 N 1
------------
  9 8 1 H 2

Now N cannot be 8 or 9. Then N = 7. However, this implies that H = 4, a contradiction.

Conclusion: the only two solutions are

  3 7 0 8 1                     3 7 0 9 1
+ 3 7 0 8 1         or        + 3 7 0 9 1
------------                  ------------
  7 4 1 6 2                     7 4 1 8 2

Of course, we can solve it by Java. Here is the code, whose file name is Main.java and belongs to package wrong_right.

package wrong_right;

/**
 * Main
 */
public class Main {

    public static void main(String... args) {
        long time = System.currentTimeMillis();
        System.out.println("");
        Main.solve();
        System.out.println("Time elapsed: " + (System.currentTimeMillis() - time) + " milliseconds.\n");
    }

    public static boolean beDistinct(int... integers) {
        for (int i = 0; i < integers.length - 1; i++) {
            for (int j = i + 1; j < integers.length; j++) {
                if (integers[i] == integers[j]) {
                    return false;
                }
            }
        }
        return true;
    }

    public static void solve() {
        // The variables are W, R, O, N, G, I, H, T, where O = 0.
        for (int w = 0; w <= 9; w++) {
            for (int r = 0; r <= 9; r++) {
                for (int n = 0; n <= 9; n++) {
                    for (int g = 0; g <= 9; g++) {
                        for (int i = 0; i <= 9; i++) {
                            for (int h = 0; h <= 9; h++) {
                                for (int t = 0; t <= 9; t++) {
                                    if (!Main.beDistinct(w, r, 0, n, g, i, h, t)) {
                                        continue;
                                    } else {
                                        if (2 * (10000 * w + 1000 * r + 10 * n + g) == 10000 * r + 1000 * i + 100 * g
                                                + 10 * h + t) {
                                            System.out.println(String.format("  %d %d %d %d %d", w, r, 0, n, g));
                                            System.out.println(String.format("+ %d %d %d %d %d", w, r, 0, n, g));
                                            System.out.println("------------");
                                            System.out.println(String.format("  %d %d %d %d %d", r, i, g, h, t));
                                            System.out.println("");
                                        }
                                    }
                                }
                            }
                        }
                    }
                }
            }
        }
    }

}
